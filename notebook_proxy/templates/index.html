This proxy creates a notebook in Google Collab from a template. It creates a notebook, stores it as a github gist then opens the notebook in google collab. The parameters passed are identifiers from the NSF Earthcube GeoCodes services. The earthcube utilites.py (ref to github and pypi) can be used to retrieve, JSON-LD files based on Dataset Science on Schema, extracted information such as downloads and descriptions of use in notebooks.
/mknb
  • url - url of the resource
  • ext - encoding format of the resource.
  • urn - urn geocodes urn. While passed url encoded, a custom encoding for a hash is needed: <hash> to replace a #
  • template -- name of the template file to render. (optional: default: template.ipynb)
Note about url, urns, ext:
A hash in the name of the item causes the rest of the url passed to be ignored. To avoid this, we custom encode '#' as <hash>
so:

`ext=http://linked.earth/ontology/core/1.2.0/index-en.html#Dataset`

becomes:

`ext=http://linked.earth/ontology/core/1.2.0/index-en.html<hash>Dataset`
